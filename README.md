# Saraya-Study-Timer

Pomodoro Timer

A simple Pomodoro Timer built using Python and Tkinter to help improve focus and productivity using the Pomodoro Technique.

Features
    •    Start a Pomodoro work session
    •    Automatic countdown timer
    •    Reset button to restart the timer
    •    Visual timer displayed on a tomato graphic
    •    Simple graphical user interface

What is the Pomodoro Technique?

The Pomodoro Technique is a time management method that alternates focused work sessions with short breaks.

Typical cycle:
    1.    Work for 25 minutes
    2.    Take a 5 minute break
    3.    Repeat for four sessions
    4.    Take a longer break

This timer automates that workflow.
